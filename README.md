# SGSA v2 — Aquaa

Proyecto HTML/CSS/JavaScript para capturar solicitudes de almacén.

## Probar
Abre `index.html` en el navegador. En modo demostración genera un folio, pero no guarda datos.

## Conectar con Google Sheets
1. Crea un Google Sheet.
2. Abre Extensiones > Apps Script.
3. Copia `GoogleAppsScript.gs`.
4. Sustituye `SPREADSHEET_ID`.
5. Publica como aplicación web.
6. Copia la URL en `js/app.js`, dentro de `CONFIG.appsScriptUrl`.

## Catálogo
Edita `js/catalogos.js` y sustituye los productos de ejemplo por el catálogo real.

## Importante
La imagen adjunta se valida, pero esta versión todavía no la sube a Google Drive.
